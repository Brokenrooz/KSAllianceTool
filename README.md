# Kingshot Alliance Tool (KSLookup)

A desktop tool for tracking alliance rosters and kingdom-wide member rankings for Kingshot:
- Roster management by alliance (TC/TG, Power, Mystic, Notes)
- Kingdom-wide leaderboard (aggregates all stored profiles)
- Fast updates by GID or Name
- Optional English alias field for readability

## Data & Fair Use
This tool is designed to store and organize **publicly visible in-game information** (e.g., roster-visible stats).
It does **not** access private account data, and it does **not** provide gameplay automation.

Users are responsible for complying with Kingshot’s Terms of Service and any applicable rules.
